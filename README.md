# Focusly Growth Model — Bottom-Up SaaS Analysis
## Project Overview
A complete bottom-up growth model built for Focusly — a freemium productivity app for university students. This model demonstrates how growth teams at FAANG companies analyse user acquisition, retention and revenue.
## What This Model Contains

 Acquisition Funnel    — 5 stage funnel from impressions to revenue
 Retention Curves      — Day 0 to Day 90 user retention analysis
 Revenue Projections   — 12 month forward looking revenue model
 Sensitivity Analysis  — 3 scenario analysis (worst, base, best case)
 Executive Dashboard   — All key metrics in one view

## Key Findings

Out of 100,000 impressions only 12 users convert to paid
Day 1 retention is the highest leverage point for revenue growth
Best case scenario generates 180x more revenue than worst case
Improving CTR from 3% to 5% has highest single metric revenue impact
